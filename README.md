# ⚙️ Student Task Manager API

A RESTful backend API developed using Node.js and Express.js for managing student tasks efficiently.

## 📖 Project Overview

This project provides backend services for a Student Task Manager application. It exposes REST API endpoints that allow users to create, retrieve, update, and delete tasks.

The API follows RESTful architecture principles and returns data in JSON format.

---

## 🚀 Features

### ✅ RESTful API Design

Supported Endpoints:

| Method | Endpoint       |
| ------ | -------------- |
| GET    | /api/tasks     |
| POST   | /api/tasks     |
| PUT    | /api/tasks/:id |
| DELETE | /api/tasks/:id |

---

### ✅ Data Validation

The API validates:

* Task title is required
* Proper request format
* Invalid requests handling

---

### ✅ JSON Responses

Returns structured JSON responses with:

* Success messages
* Error messages
* Task information

---

### ✅ HTTP Status Codes

Implemented status codes:

* 200 OK
* 201 Created
* 400 Bad Request
* 404 Not Found
* 500 Internal Server Error
